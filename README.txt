
* Clone this repository and change it's name from "static-site-template" to the project name


RUN THE FOLLOWING IN THE COMMAND LINE TO ENABLE SASS & GULP

npm i
npm install --global gulp-cli           (sudo npm install --global gulp-cli (MacOS))
npm install --save-dev gulp
npm install --save-dev gulp-autoprefixer
npm install sass gulp-sass --save-dev
npm install gulp-clean-css --save-dev
npm install --save-dev gulp-babel @babel/core @babel/preset-env
npm install --save-dev gulp-concat


* Run 'gulp' in the command line
* Right click on index.html 
* Select 'Open with Live Server
